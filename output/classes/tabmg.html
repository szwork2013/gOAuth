api/tagmg/alltags

(
  • from
  • size
)

Defined in controller/api/tagmg.js:46

分页获取标签列表
请求类型: GET
请求类型: application/json
返回结构:
{ "errcode": "0", "errmsg": "", "data":{}, "extention":{} }
data 参数名说明:
id 标签ID，GUID
name 标签名称
desc 标签说明
status 是否启用，0代表禁用，1代表启用
count

Parameters:

  • from Number

    列表起始位置，1代表第一条数据

  • size Number

    返回条数，建议10

Returns:

返回架构中的 data 说明，data 是一个数组结构

Example:

输入样例
{
    "name":"美男子",
    "status":"1",
    "from": 1,
    "size": 10
}

返回样例：
{
    "count":"",
    "result": [{
        "id":"3b7c2d81-aa23-4025-85a7-44c28a472718",
        "name":"美男子",
        "desc":"给所有美男子打的标签",
        "status":"1"
    }]
}
返回码说明
0        创建或修改成功
40001    创建或修改失败
